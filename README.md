This will follow the Odin Project's Mini Message Board project. This will be a small, simple project utilizing a variety of features of Express.

This project uses Express and EJS.

PostgreSQL is used with node-postgres. The database is hosted through Neon.
If you try to run the code yourself, you will need to set a connection to a database environment variable.

CONNECTION_STRING='<connection>'

It will demonstrate a basic capability to create a backend using Node.

There is a deployment available at https://odin-mini-message-board-83st.onrender.com
